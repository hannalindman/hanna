API Toys Decutmentation

GET

Search for a specific item, for example, doll: "https://toys2.herokuapp.com/toys?s=car"
to get specific page: "https://toys2.herokuapp.com/toys?page=1"
Get a certain number of items on the page, for example 5: "https://toys2.herokuapp.com/toys?perPage=5"
Sort by name or information: For example Train: "https://toys2.herokuapp.com/toys?sort=train"
Search for an item by minimum price, for example 10: "https://toys2.herokuapp.com/toys?min=10"
Search for an item by maximum price, for example 100: "https://toys2.herokuapp.com/toys?max=100"

POST

To add an item: "https://toys2.herokuapp.com/toys"

PUT

To edit an item: "https://toys2.herokuapp.com/toys/(The id of the item)"

DELETE

To delete an item: "https://toys2.herokuapp.com/toys/(The id of the item)"
